# Projeto -- Busca em Largura (BFS) em Python

Este projeto implementa algoritmos de **Busca em Largura (BFS)**
utilizando uma estrutura de grafo baseada em **lista de adjacência**.

## Funcionalidades

### 1. BFS Padrão

Percorre o grafo visitando primeiro os vértices mais próximos,
retornando a **ordem de visita**.

### 2. Menor Caminho (BFS)

Identifica o menor caminho entre dois vértices em um grafo **não
ponderado**, utilizando a lógica do BFS para rastrear pais.
